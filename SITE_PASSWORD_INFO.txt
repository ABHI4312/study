🔒 SITE PASSWORD PROTECTION
============================

The website is now protected with a password gate that appears when anyone opens the site.

🔑 DEFAULT PASSWORD:
-------------------
Abhiya@1328


📝 HOW IT WORKS:
----------------
1. When someone opens the website, they see a beautiful password dialog
2. They must enter the correct password to access the site
3. Password is stored in sessionStorage (expires when browser closes)
4. After 3 failed attempts, a hint is shown


🛠️ HOW TO CHANGE PASSWORD:
---------------------------
Open this file:
frontend/src/components/SitePasswordGate.jsx

Find this line (around line 11):
const SITE_PASSWORD = 'Abhiya@1328';

Change it to your desired password:
const SITE_PASSWORD = 'YourNewPassword123';


✨ FEATURES:
-----------
- Beautiful animated dialog with floating hearts
- Show/hide password toggle
- Shake animation on wrong password
- Hint after 3 failed attempts
- Session-based (password required again after closing browser)
- Fully responsive design


🎨 CUSTOMIZATION:
-----------------
You can customize the dialog in:
frontend/src/components/SitePasswordGate.jsx

- Change colors
- Modify animations
- Update text/messages
- Add more security features


💡 SECURITY NOTES:
------------------
- This is CLIENT-SIDE protection (basic privacy)
- Password is visible in source code
- For production, consider backend authentication
- Good for keeping casual visitors out
- Not suitable for highly sensitive data
